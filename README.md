# Teensy_Flasher
Projekt zur Kabellosen Steuerung eines Roboter ( mit Teensy 4.0 Steuerung ) durch einen Raspberry Pi Zero W über eine Public Website unter (bodenseekoalas.markdorf-robotics.de). Das projekt dient zur teilnahme am Robocup Junior 2026. Es soll erreicht werden ,dass per Serielle befehle der Teensy gesteuert wird und zusätzlich Software geladen werden.
